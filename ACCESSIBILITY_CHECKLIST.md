# רשימת בדיקות נגישות (Checklist)

## ✅ בדיקות בסיסיות (חובה)

### תמונות
- [ ] כל תמונה יש לה `alt` attribute
- [ ] תמונות דקורטיביות: `alt=""`
- [ ] תמונות תוכן: `alt` מתאר את התוכן בעברית

### כותרות
- [ ] כל עמוד יש לו רק H1 אחד
- [ ] מבנה כותרות לוגי: H1 → H2 → H3
- [ ] אין דילוג על רמות (לא H1 → H3)

### ניגודיות צבעים
- [ ] יחס ניגודיות מינימלי 4.5:1 לטקסט רגיל
- [ ] יחס ניגודיות מינימלי 3:1 לטקסט גדול (18pt+)
- [ ] בדיקה עם כלי ניגודיות (WebAIM Contrast Checker)

### ניווט מקלדת
- [ ] כל האלמנטים האינטראקטיביים נגישים דרך TAB
- [ ] Focus outline נראה בבירור על כל האלמנטים
- [ ] אין "מלכודות מקלדת" (keyboard traps)

### טפסים
- [ ] כל שדה טופס יש לו `<label for="id">` מתאים
- [ ] שדות חובה מסומנים בבירור
- [ ] הודעות שגיאה נגישות לקוראי מסך
- [ ] `aria-invalid` ו-`aria-describedBy` לשדות עם שגיאות

### ARIA
- [ ] `role="navigation"` לתפריטים
- [ ] `aria-label` לכפתורים ללא טקסט
- [ ] `aria-expanded` לתפריטים נפתחים
- [ ] `role="main"` לתוכן הראשי
- [ ] `role="contentinfo"` לפוטר

### קישורים
- [ ] כל קישור יש לו טקסט מתאר
- [ ] אין קישורים "לחץ כאן" או "קרא עוד" ללא הקשר
- [ ] קישורים חיצוניים: `target="_blank" rel="noopener"`
